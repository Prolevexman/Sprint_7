# Автоматизация тестирования API

Проект автоматизации тестирования API для Яндекс Самокат.  
Включает тесты для курьеров (`Courier`) и заказов (`Order`) с позитивными и негативными сценариями, параметризацией и генераторами данных.

## Требования
- Java 11+
- Maven
- Allure
- Доступ к тестовому API


## Запуск тестов

### Через Maven
```bash
# Все тесты
mvn clean test 
```
### Генерация отчёта Allure
```bash
mvn allure:serve

```
